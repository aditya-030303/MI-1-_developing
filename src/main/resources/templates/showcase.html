🚀 What is RiddleVerse?

A riddle-based user authentication and engagement platform. Instead of passwords, users answer riddles to log in. Once inside, they can submit, view, and solve riddles from others!

🧩 Key Features

  • 🔐 Riddle-based login system (used every time)
  • 👤 User profile creation after login
  • ✍️ Add and store riddles with user attribution
  • 🌐 Explore riddles from other users
  • ✅ Answer riddles and receive feedback

🛠️ Tech Stack

📖 Backstory

Born out of the idea that login shouldn't be boring, RiddleVerse combines brain teasers with identity verification. It's a fun, creative way to interact with users — no passwords, just clever answers.
